# Image Segmentation Project

This project demonstrates an end-to-end image segmentation workflow built in Python using TensorFlow, OpenCV, and supporting data science libraries. It covers dataset preparation, preprocessing, model training, and evaluation in a reproducible notebook-based format.

It is designed to show practical skills in computer vision, experiment structure, and working with real-world image data. The repository can also serve as a clear reference implementation for segmentation experiments.

## Contents

- `Thakur_Bhavin_23079699.ipynb` - main notebook
- `requirements.txt` - Python dependencies
- `RMDS_segmentation_dataset_ADE20K_350/` - dataset and generated outputs

## Highlights

- Image segmentation pipeline from data loading to evaluation
- Notebook workflow suitable for experimentation and presentation
- Local dataset structure with saved outputs for repeatable results

## Setup

```bash
pip install -r requirements.txt
```

## Usage

Open the notebook in Jupyter or VS Code and run the cells in order.

## Notes

- The dataset is stored locally in the repository folder.
- Generated masks and outputs are saved under the dataset directory.
